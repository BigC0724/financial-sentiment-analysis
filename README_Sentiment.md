
# Financial Sentiment Analysis Using FinBert

## Overview
A financial news sentiment analysis system using FinBert
a pre-trained NLP model specifically trained on financial
text. Analyzes headlines and classifies them as positive
negative or nutral with confidence scores.

## Results
- Overall Accuracy: 77%
- Negative Sentiment Accuracy: 82%
- Positive Sentiment Accuracy: 72%
- Dataset: 100 financial news headlines

## Key Finding
FinBert outperformed traditional TF-IDF models by 52
percentage points (77% vs 25%) demonstrating the power
of domain specific pre-trained language models in
financial text analysis

## Tools used
- Python
- FinBert (ProsusAI/finbert)
- HuggingFace Transformers
- Pandas, NumPy, Matplotlib

Real world financial headlines covering:
